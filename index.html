The Honest Assessment
What the system history tells us — before we talk about solutions
⚠
The original AV system in this building was not installed by Fathom Media. By every account from every stakeholder, it never worked correctly from day one. Fathom Media has been troubleshooting, patching, and repairing what was inherited. That era ends here — with a plan, not another patch.

Platform Strategy
Why Q-SYS — and Why Now
One unified platform replaces the fragmented Crestron + Biamp environment

The problem at THQ is not individual component failures — it is a fragmented platform that was never designed to work as one system. The solution is a migration to Q-SYS: unified audio processing, room control, and campus automation across every space.

Current: Crestron + BiampProposed: Q-SYS by QSC
Platform UnityTwo systems, two programming environments, two support contractsOne platform — audio, video, and control in a single design file
Remote SupportSeparate XiO Cloud + Biamp access — complex, expensive, rarely effectiveQ-SYS Reflect — native cloud monitoring, Fathom Media sees alerts first
Programming ChangesCertified Crestron programmer required for every change — $2,000–$3,500/visitRemote changes by Fathom Media, often same day, no truck roll
Power Cycle RecoveryDMPS loses all routing — manual re-routing required after every outageQ-SYS auto-restores system state — rooms recover automatically
ScalabilityEach new room = new hardware + programming + Biamp configurationNew rooms added as Q-SYS peripherals on existing network
End-User ExperienceUsers must understand system state — failures not self-evidentSimple, consistent panels — complexity invisible to users
Leadership TransitionsEach new person requires retraining on a complex, inconsistent systemSame interface in every room — one training, works everywhere
🎯
Q-SYS Core 24f — the current-generation successor to the Core 110f (EOL Dec 2025). Provides 24 analog I/O channels, USB-C AV bridging, GPIO + RS232 control, 4 network ports, and native Teams Rooms integration — all in 1RU. This is the processing backbone of every upgraded room at THQ.

🔌

WattBox Smart PDU

Installed in every rack. Remote outlet-level power cycling. Automated reboot scheduling. Current monitoring and alerting. Directly addresses the C-Box power supply failures in the boardrooms.

💻

Remote Access PC (Intel NUC)

Small-form-factor PC in every primary rack. When THQ staff grant access, Fathom Media connects directly to the local network. Full system visibility, no VPN dependency. Most issues resolved without rolling a truck.

☁

Q-SYS Reflect Cloud Monitoring

24/7 monitoring of all Q-SYS cores and peripherals. Real-time alerts on system health, component failures, and anomalies — often before users are aware. Dashboard visible to THQ IT and Fathom Media simultaneously.

📋

Network Documentation

Complete IP map, MAC address registry, VLAN assignments, and switch port documentation — maintained by Fathom Media and updated after every service visit.

💰
The ROI is direct: A single on-site visit for a routing issue that could be resolved remotely costs $800–$1,200 in labor. The WattBox + Remote PC + Q-SYS Reflect stack pays for itself with the first avoided truck roll.

The Partnership Model
5-Year Service Level Agreement
Not an add-on. The model under which Fathom Media operates from day one of Phase 1.

The SLA is how THQ gets the proactive, ongoing support they clearly need — and it is how Fathom Media builds a relationship defined by results, not reactive service calls. Every system installed in Phase 1 is covered from the day it goes live.

Remote Monitoring
24/7 Q-SYS Reflect + WattBox monitoring across all covered systems. Fathom Media receives alerts before users notice a problem.
Remote Support
Priority response within 4 business hours for remote-resolvable issues. Majority of problems solved without a site visit.
On-Site Response
Guaranteed on-site response within 24–48 hours for issues requiring physical presence.
Preventive Maintenance
Bi-annual on-site visits — firmware updates, cable inspection, cleaning, documentation refresh.
Annual Review
Written annual report: performance metrics, incident history, end-of-life horizon, and recommendations delivered to THQ stakeholders.
Staff Training
Annual training refresher — all spaces, all panels, all troubleshooting procedures for THQ staff.
Warranty Management
Fathom Media manages all manufacturer warranty claims and RMA logistics on covered equipment.
Leadership Onboarding
System orientation for every incoming officer or senior staff member. No institutional knowledge lost on personnel changes — ever.
Software & Firmware
Q-SYS updates and firmware maintenance included. No surprise programming invoices for standard updates.
🏛
The leadership onboarding provision is specifically designed for the THQ environment. Officer and staff transitions are frequent. Fathom Media ensures the AV system never feels foreign to anyone in the building, regardless of when they arrived.
